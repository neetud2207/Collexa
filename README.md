# 🎓 Collexa – College Discovery & Decision Platform

Collexa is a modern full-stack college discovery platform designed to help students explore, compare, and shortlist colleges efficiently.

Built using **Next.js, TypeScript, Tailwind CSS, MongoDB, and REST APIs**, the platform focuses on providing a smooth user experience with decision-making features for students.

# ✨ Features

## 🔍 College Listing + Search
- Dynamic college listing page
- Search colleges by name
- Filter by:
  - Location
  - Course
- Responsive college cards
- Real-time UI updates

---

## ⚖️ Compare Colleges
- Select multiple colleges
- Compare:
  - Fees
  - Ratings
  - Placement %
  - Location
  - Courses
- Helps students make informed decisions

---

## 🧠 Predictor Tool
- Input exam rank
- Predict suitable colleges
- Rule-based admission prediction logic

---

## ❤️ Saved Colleges
- Save favorite colleges
- Persistent storage using localStorage
- Personalized browsing experience

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/neetud2207/Collexa.git
```

## Navigate to Project

```bash
cd Collexa/client
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
